1. **promoSystem** - Система промокодов

2. **statistics** - Статистика и аналитика

### 🔄 Миграция

- Старая версия бота сохранена как `bot_fixed.js`
- Все данные полностью совместимы между версиями
- Запуск старой версии: `npm run old`
- Запуск новой версии: `npm start`

### 📁 Новая структура

```
kvantik-bot2/
├── bot.js              # Новый основной файл
├── core/               # Ядро системы
│   └── moduleLoader.js # Загрузчик модулей
├── modules/            # Директория модулей
├── README.md           # Основная документация
└── MODULES_GUIDE.md    # Руководство по модулям
```

### 🚀 Запуск

```bash
# Установка
npm install

# Настройка
cp .env.example .env
# Отредактируйте .env

# Запуск
npm start
```

### ⚠️ Важные изменения

1. **Новый главный файл**: теперь `bot.js` (вместо `bot_fixed.js`)
2. **Модули**: добавлена папка `core/` с системой загрузки
3. **Конфигурация**: все настройки через `.env` и `config/config.js`
4. **Логирование**: новый формат с контекстом `logger.info(context, message, meta)`

### 📊 Статистика

- Строк кода: ~1500 (новая архитектура)
- Модулей: 3 встроенных + примеры
- Утилит: 6 (logger, storage, validator, backup, monitoring, exporter)
- Документация: 3 файла (README, MODULES_GUIDE, CHANGELOG)
